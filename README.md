# 📋 Desafio Prático: Formulário de Inscrição / Convite (Rocketseat)

Status do Projeto: Concluído 🚀

## 📖 Sobre o Projeto
Este projeto foi desenvolvido como um desafio prático da trilha de desenvolvimento Fullstack da **Rocketseat**. O principal objetivo foi consolidar conhecimentos na **criação, estruturação e estilização de formulários web**, focando em boas práticas, semântica HTML e experiência do usuário (UX).

### 💡 Diferencial do Desenvolvimento
O grande marco deste projeto foi a metodologia de execução: **desenvolvido 100% de forma autônoma**. 
* **Zero vídeos de passo a passo:** A aplicação foi construída do zero sem acompanhar resoluções ou tutoriais em vídeo.
* **Figma to Code:** O desenvolvimento foi baseado exclusivamente na análise do layout proposto no Figma, exigindo autonomia na tomada de decisões técnicas, resolução de problemas estruturais e fidelidade visual.

---

## 🛠️ Aprendizados e Conceitos Exercitados
* **HTML5 Semântico:** Estruturação correta de formulários utilizando tags apropriadas (`<form>`, `<label>`, `<input>`, `<fieldset>`, `<legend>`, `<select>`).
* **Estilização Avançada de Elementos de Formulário:** Customização de estados de inputs (`:focus`, `:valid`, `:invalid`, `:disabled`), tratamento de placeholders e reset de estilos padrão do navegador.
* **Design Responsivo:** Garantia de que o formulário seja perfeitamente utilizável tanto em dispositivos móveis quanto em desktops.
* **Resolução Autônoma de Problemas:** Tradução de elementos visuais do Figma (espaçamentos, sombras, tipografia e cores) diretamente para o código.

---

## 🚀 Tecnologias Utilizadas
* **HTML5** (Estruturação semântica)
* **CSS3** (Estilização, Flexbox/Grid, Variáveis CSS e Responsividade)
